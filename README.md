## Install
Store your API keys in `key.ev`:
```
OPENAI_API_KEY=""
ANTHROPIC_API_KEY=""
```
Then install the environment:
```
conda create -n w4s python=3.11
conda activate w4s
pip install -r requirements.txt
```
